## Docker Commands
| فرمان          | توضیح                        |
| -------------- | ---------------------------- |
| `docker build` | ساخت image از روی Dockerfile |
| `docker run`   | اجرای کانتینر                |
| `docker ps`    | نمایش کانتینرهای فعال        |
| `docker stop`  | توقف یک کانتینر              |
| `docker exec`  | اجرای دستور داخل کانتینر     |
| `docker logs`  | مشاهده خروجی یک کانتینر      |
